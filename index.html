Projects

Pensieve

A personal blogging website mirroring a diary allowing users to pen down memories and thoughts and store them in a secure manner. Implemented using Apache Tomcat Server and JSP and deployed on heroku. The standout feature was the selective tagging of users in memories involving a private group of users. The music tracks provided in the writing area cater to the different moods of the writer.

Crawl-Bot

A Lightweight Machine Learning System in a container based virtual environment (Docker) enabling the extraction of data from HTML/XHTML pages using specific Python packages. Prediction of classes, provided with certain input values accomplished using Decision Trees. Projection of statistics after data mining implemented using Jupyter Notebook.

Book(a)Room

Simulation of Library Room Booking System for different time slots and locations using Ruby on Rails deployed on Heroku.
